## Deployment

To run this script use this steps

```bash
  git clone https://github.com/AnkanSaha/Deploy-Node-App-Enviroment-Config.git
```
Go to the project directory

```bash
  cd Deploy-Node-App-Enviroment-Config
```

Run the script

```bash
  ./deploy.sh
```
